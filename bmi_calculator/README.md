BMI Calculator

Project Description

This is a Python GUI-based BMI (Body Mass Index) Calculator developed as part of the Oasis Infobyte 
Python Programming Internship.

The application allows users to enter their name, weight, and height, calculates their BMI,
classifies it into a health category, stores the data in a CSV file, displays previous records,
and shows a BMI graph using Matplotlib.

Features

- GUI built using Tkinter
- Calculate BMI
- BMI Categories:
  - Underweight
  - Normal
  - Overweight
  - Obese
- Color-coded BMI result
- Input validation
- Error handling
- Multi-user support
- Save records in a CSV file
- View BMI history
- Display BMI graph using Matplotlib

Technologies Used

- Python
- Tkinter
- CSV
- Matplotlib

How to Run

1. Install Python.
2. Install the required libraries.
3. Run the "bmi_calculator.py" file.
4. Enter your name, weight, and height.
5. Click Calculate BMI.

Project Structure

- "bmi_calculator.py" – Main Python program
- "bmi_records.csv" – Stores BMI history
- "README.md" – Project documentation

Outcome

This project demonstrates the use of Python GUI programming, file handling,
data visualization, and exception handling to build a simple and user-friendly BMI Calculator.
